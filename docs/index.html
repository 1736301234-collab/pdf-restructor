快速开始

三步完成安装

1

安装Python

确保已安装 Python 3.8+

$ python --version
2

克隆仓库

下载项目代码

$ git clone https://github.com/yourusername/pdf-restructor.git
$ cd pdf-restructor
3

安装依赖

安装必要依赖

$ pip install -r requirements.txt

使用方法

🎯 命令行使用

# 基本转换
$ python test_with_user_pdf.py book.pdf

# 启用OCR（扫描版PDF）
$ python test_with_user_pdf.py scan.pdf --ocr

# 自定义输出
$ python test_with_user_pdf.py book.pdf -o output.epub

🖥️ 图形界面

$ python gui_converter.py
