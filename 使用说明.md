# PDF转EPUB工具 - 使用说明
写在前面，ocr环境安装有些麻烦，所以测试未通过，但是已经可以建议使用没有偶承认还有ai的版本进行转变格式，但是普通版本效果还可以，适合不能在云端转变格式的人去使用。
## 📥 第一步：准备环境

### 1. 安装Python
确保你的电脑已安装 Python 3.8 或更高版本：
```bash
# 打开命令提示符(CMD)或PowerShell，输入：
python --version
# 应该显示 Python 3.8.x 或更高版本
```

### 2. 安装必要依赖
打开命令提示符(CMD)，进入工具所在目录，运行：
```bash
cd 你的工具目录路径\pdf_restructor

pip install -r requirements.txt
```

### 3. 可选但推荐的依赖
```bash
# 中文OCR支持（强烈推荐）
pip install paddleocr

# 或 EasyOCR
pip install easyocr

# AI语义分析
pip install spacy
python -m spacy download zh_core_web_sm
```

---

## 🚀 第二步：运行转换

### 方法A：图形界面（最简单）

```bash
cd pdf_restructor目录
python gui_converter.py
```

然后：
1. 点击"浏览..."选择你的PDF文件
2. 选择输出位置（可选，默认保存到同一目录）
3. 勾选需要的选项（OCR建议勾选）
4. 点击"开始转换"
5. 等待完成

### 方法B：命令行（推荐）

```bash
# 基础转换（自动检测）
python test_with_user_pdf.py "E:\你的路径\文件.pdf"

# 启用OCR（扫描版PDF推荐）
python test_with_user_pdf.py "E:\你的路径\文件.pdf" --ocr

# 启用AI分析
python test_with_user_pdf.py "E:\你的路径\文件.pdf" --ocr --ai

# 指定输出位置
python test_with_user_pdf.py "E:\你的路径\文件.pdf" --output "D:\output\结果.epub"
```

### 方法C：拖拽方式（Windows）

创建一个批处理文件 `转换.bat`：
```bat
@echo off
python "%~dp0test_with_user_pdf.py" "%1" --ocr --ai
pause
```

然后把PDF文件拖到这个bat文件上即可自动转换。

## 📂 第三步：查找输出文件

转换完成后，EPUB文件会保存在：

**默认位置**：
- 如果PDF在：`E:\文档\book.pdf`
- 输出在：`E:\文档\book_converted.epub`

**你指定了输出位置**：
- 就在你指定的位置

## ⚠️ 常见问题

### Q: 提示找不到依赖？
```bash
# 重新安装依赖
pip install --upgrade -r requirements.txt
```

### Q: OCR失败？
```bash
# 单独安装OCR支持
pip install paddleocr -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 转换失败？
1. 检查PDF文件是否完整
2. 如果是加密PDF，需要提供密码
3. 查看日志中的错误信息

### Q: 中文显示乱码？
- 这是正常的，因为EPUB使用标准字体
- 在电子书阅读器中会自动选择合适的中文字体

## 🔧 高级用法

### 批量转换
```bash
python -c "
import os
from core.enhanced_pipeline import convert_pdf_to_epub_enhanced

for file in os.listdir('E:\\pdf_files'):
    if file.endswith('.pdf'):
        convert_pdf_to_epub_enhanced(
            f'E:\\pdf_files\\{file}',
            f'E:\\output\\{file[:-4]}.epub',
            enable_ocr=True
        )
"
```

### 自定义样式
编辑 `core/generator.py` 中的 `EPUBConfig` 类：
```python
config = EPUBConfig(
    font_size=18,        # 字体大小
    line_height=2.0,     # 行高
    font_family="SimSun" # 字体
)
```
